# Github-colaborativo
Repositorio colaborativo en Python
El objetivo es colaborar en un proyecto sencillo en Python con tres módulos separados, donde cada módulo tenga una funcionalidad diferente

1. operaciones_matematicas.py
  Contiene funciones para realizar suma, resta, multiplicación y división.
2.cuento.py
  Imprime un cuento corto en pantalla.
3.conversor_unidades.py
   Convierte temperaturas entre Celsius y Fahrenheit.
